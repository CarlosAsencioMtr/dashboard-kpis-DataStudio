# Dashboard de Looker Studio

Este dashboard fue desarrollado para visualizar indicadores empresariales a partir de datos procesados mediante Google Sheets y Google Apps Script.

## Actualización

El dashboard se actualiza automáticamente cada 5 minutos, permitiendo consultar información en tiempo casi real.

## Nota de confidencialidad

Por motivos de confidencialidad, el enlace público al dashboard no se incluye en este repositorio.

En su lugar, se pueden incluir capturas de ejemplo en la carpeta `images/`.
