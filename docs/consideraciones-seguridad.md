# Consideraciones de Seguridad y Confidencialidad

Por motivos de seguridad y confidencialidad, este repositorio no incluye datos reales del ERP empresarial ni información sensible de la organización.

## Información no incluida

Este repositorio no contiene:

- Credenciales
- Contraseñas
- Tokens
- URLs privadas
- IDs reales de Google Sheets
- Correos electrónicos internos
- Datos reales de clientes
- Información financiera confidencial
- Reportes internos privados

## Uso de datos de ejemplo

Los datos incluidos en este repositorio son ficticios o anonimizados. Su propósito es demostrar la lógica del proceso ETL, la estructura del flujo de datos y la integración con herramientas de Business Intelligence.

## Buenas prácticas aplicadas

- Uso de datos de muestra
- Documentación técnica del proceso
- Separación entre lógica, datos y visualizaciones
- Protección de información sensible
