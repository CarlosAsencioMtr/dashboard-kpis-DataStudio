# Diccionario de Datos

Este documento describe una estructura de datos de ejemplo utilizada para representar la información procesada en el flujo ETL.

| Campo | Descripción | Tipo de dato |
|---|---|---|
| fecha | Fecha del registro | Fecha |
| cliente | Cliente o entidad relacionada | Texto |
| categoria | Categoría del producto, servicio o proceso | Texto |
| producto | Producto o servicio registrado | Texto |
| cantidad | Cantidad asociada al registro | Número |
| monto | Valor monetario asociado | Número |
| estado | Estado del registro | Texto |

## Nota

Los campos utilizados son genéricos y no representan información confidencial de ninguna organización.
